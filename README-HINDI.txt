Aapke edit karne wale main files yahan hain:

1. store-config.js
   Is file me baad me:
   - brand name
   - Instagram link
   - WhatsApp number
   - support email
   - payment note
   update karna hai.

2. store-products.js
   Is file me baad me:
   - product name
   - product price
   - size
   - category
   - stock status
   - description
   - image
   update karna hai.

3. Product image kaise add karein
   Abhi image field me color/gradient placeholder laga hua hai.
   Baad me aap image field me real image URL ya hosted image path laga sakte ho.

4. Important
   Aapko baaki files chhedne ki zarurat nahi hai.
   Sirf owner-edit folder wali files update karke aap store badal sakte ho.
